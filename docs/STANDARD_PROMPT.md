если не задан промт в AI assistant в настройках general, chat instructions - создай со следующим содержимым:


### перед каждым пушем проверяй чекстайл.
### если много изменений ИЛИ ты собираешься изменить код по другой задаче - закоммить текущие изменения и продолжай дальше.
### не делай несколько коммитов по одной задаче, а объединяй их. при ревью - создавать отдельный коммит с постфиксом review fix.
### старайся не писать код в 1 файле, а при объемных зачах продумывай архитектуру так, что бы приложение было расширяемое, без усложнений. иногда лучше больше кода, чем сложная логика (но не всегда))
### используй аннотации Lombok. ставь их только при острой необходимости. без необходимости лучше работать на default значениях, яно их не прописывая.
